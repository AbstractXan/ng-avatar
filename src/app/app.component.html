ng-avatar

A simple ng-component for customizable avatars

by Priyansh Sangule



name

Name to be used for initials
{{name}}
image

Image path
characters

No. of characters to be taken from initials
{{characters}}
bgColor

Background Colour
textColor

Text color for the name
size

Size of avatar- 'sm' 'md' 'l' 'xl' or a number
rounded

Avatar shape
{{rounded}}
radius

Corner radius for square avatar
margin
randomColor

Randomize avatar color
{{randomColor}}
label

Label text
labelBgColor

Label's background color
labelTextColor

Label's text colour
active
{{active}}
uploadable

Camera image on hover
{{uploadable}}

Examples

<app-avatar name="AbstractXan" characters=2 bgColor="blue" radius=12 ></app-avatar>
<app-avatar name="John Hamish Watson" characters=3 randomColor=true ></app-avatar>
<app-avatar name="Albert Einstein" bgColor="orange" [rounded]=true [uploadable]=true></app-avatar>
